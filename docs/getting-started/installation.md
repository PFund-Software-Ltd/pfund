## Using [Poetry](https://python-poetry.org) (Recommended)
```bash
poetry add pfund

# update to the latest version:
poetry update pfund
```


## Using Pip

```bash
pip install pfund

# install the latest version:
pip install -U pfund
```


## Checking your installation
```bash
$ pfund --version

pfund, version 0.0.1.dev4
```
